# FaceVisitor REST API 
Spring boot JPA 기반 REST API 서버 

## Features
- 고객의 얼굴인식을 통한 빠른 **가입 및 로그인**
- 고객의 얼굴을 기반으로 한 개인화된 **상품 추천**
- QR코드를 통한 **셀프결제**
